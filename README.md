# Social-Media-Sentimental-Analysis-Dashboard

## Project Objective
The objective of this project is to analyze social media data to understand public sentiment (positive, negative, and neutral) toward topics, products, or services, helping businesses make data-driven decisions.

## Dataset Used
- <a href="https://github.com/bk9991342-cmyk/Social-Media-Dashboard/blob/main/Posts.xlsx">Dataset-1</a>
- <a href="https://github.com/bk9991342-cmyk/Social-Media-Dashboard/blob/main/Users.xlsx">Dataset-2</a>

## Process
-   Collected and cleaned social media data (posts, hashtags, dates, sentiment scores).
-  Processed data using SQL/Excel for filtering, aggregation, and sentiment classification.
-  Imported cleaned data into Power BI.
-  Created interactive visuals such as sentiment distribution, trend analysis, and top hashtags.
-  Applied slicers for date, sentiment type, and topic analysis.
